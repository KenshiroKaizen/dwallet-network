Use one of the following commands for [Homebrew](https://brew.sh/) (MacOS, Linux, or Windows Subsystem for Linux) or [Chocolatey](https://chocolatey.org/) (Windows) to install Ika.

<Tabs groupId="operating-systems">

<TabItem label="Homebrew" value="mac">

```bash
brew install ika
```

</TabItem>

<TabItem label="Chocolatey" value="win">

```shell
choco install ika
```

Find more [versions of Ika for Windows](https://community.chocolatey.org/packages/ika) on the Chocolatey community website.

</TabItem>

</Tabs>
