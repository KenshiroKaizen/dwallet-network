## Ika CLI

Interact directly with Ika networks and its features using the Ika command line interface (CLI). The CLI is divided into separate base commands that target a specific set of features.

<Cards>
<Card title="Ika Client CLI" href="references/cli/client">
Create a client on a Ika network to generate addresses, access networks, and more with the Ika Client CLI.
</Card>
<Card title="Ika Client PTB CLI" href="references/cli/ptb">
Build, preview, and execute programmable transaction blocks directly from your terminal with the Ika Client PTB CLI.
</Card>
<Card title="Ika Move CLI" href="references/cli/move">
Access Ika Move functions on chain using the Ika Move CLI.
</Card>
</Cards>
