---
title: Ika and Community SDKs
description: Collection of SDKs and utilities for developing on Ika using various programming languages.
---

Ika provides developer kits that act as wrappers for the Ika API. The Ika community broadens the code coverage with its own set of developer kits targeting the Ika blockchain. 

## Ika SDKs

<Cards>
<Card title="dApp Kit" href="https://sdk.mystenlabs.com/dapp-kit">
    A web frontend SDK that interacts with the Ika API. It is available as an NPM package.
</Card>
<Card title="Rust SDK" href="./rust-sdk">
    SDK configuration and examples of using the Ika API with Rust, using the `ika-sdk` crate.
</Card>
<Card title="Rust SDK Auto-generated Docs" href="https://mystenlabs.github.io/ika/ika_sdk/index.html">
    Auto-generated documentation for the `ika-sdk` crate in the Ika repository.
</Card>
<Card title="TypeScript SDK" href="https://sdk.mystenlabs.com/typescript">
    TypeScript SDK for integrating Ika in your TS dApps.
</Card>
<Card title="zkSend SDK" href="https://sdk.mystenlabs.com/zksend">
    zkSend SDK to enable you to incorporate Stashed functionality.
</Card>
</Cards>

## Community SDKs

:::info 

While the community projects are expertly developed, their maintenance and community support vary. You might want to research a project's history and support level before committing to using its utilities. 

:::

<Cards>
<Card title="dApp Kit (Vue)" href="https://github.com/IkaFansCN/ikaue">
    Ika dApp Kit for the Vue framework.
</Card>
<Card title="Dart SDK" href="https://github.com/mofalabs/ika">
    A cross-platform Ika SDK for mobile, web, and desktop.
</Card>
<Card title="Go SDK" href="https://github.com/block-vision/ika-go-sdk">
    SDK for developing for Ika using Golang.
</Card>
<Card title="Kotlin SDK" href="https://github.com/mcxross/kika">
    Kika is a collection of Kotlin Multiplatform JSON-RPC wrapper and crypto utilities for interacting with a Ika Full node.
</Card>
<Card title="Python SDK" href="https://github.com/FrankC01/pyika">
    pyika is a Python client for developing on the Ika blockchain. 
</Card>
<Card title="Swift SDK" href="https://github.com/opendive/ikakit">
    IkaKit is a Swift SDK natively designed for developing on the Ika blockchain.
</Card>
</Cards>
