# The `@ika-io/ika/utils` package

This package contains some utilities that simplify common operations when working with the Ika
TypeScript SDK.

## Constants

A set of constants exported for common uses cases:

- `NIKA_PER_IKA`: The conversion rate for NIKA to IKA (1,000,000,000)
- `IKA_DECIMALS`: the number of decimals you must shift a NIKA value to convert it to IKA (`9`)
- `IKA_ADDRESS_LENGTH`: The number of bytes in a Ika address (32)
- `MOVE_STDLIB_ADDRESS`: The address for the Ika Move standard library
- `IKA_FRAMEWORK_ADDRESS`: The address for the Ika Framework
- `IKA_SYSTEM_ADDRESS`: The address for the Ika System module
- `IKA_CLOCK_OBJECT_ID`: The address for the `ika::clock::Clock` object
- `IKA_SYSTEM_STATE_OBJECT_ID`: The address for the `IkaSystemState` object

## Formatters

You can use the following helpers to format various values:

- `formatAddress`
- `formatDigest`
- `normalizeStructTag`
- `normalizeIkaAddress`
- `normalizeIkaObjectId`
- `normalizeIkaNSName`
- `normalizeIkaNSName`

## Validators

You can use the following helpers to validate the format of various values (this only validates that
the value is in the correct format, but does not validate the value is valid for a specific use
case, or exists on chain).

- `isValidIkaAddress`
- `isValidIkaObjectId`
- `isValidTransactionDigest`
- `isValidIkaNSName`

## Encoding

The following methods are re-exported to help with converting between commonly used encodings

- `fromHex`: Deserializes a hex string to a Uint8Array
- `toHex`: Serializes a Uint8Array to a hex string
- `fromBase64`: Deserializes a base64 string to a Uint8Array
- `toBase64`: Serializes a Uint8Array to a base64 string
